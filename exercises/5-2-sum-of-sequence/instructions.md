# 5.2 Sum of Sequence (while)

## Task

Please ask the user to input a sequence of integers. Each integer should be entered via a separate input statement. You should stop asking for more once the user enters `0`.

Please print the sum of all numbers entered this way.

## Examples

If the user inputs `1`, `3`, `2`, `1` and `0`, you should print `7`.

If the user inputs `1`, `7`, `9` and `0`, you should print `17`.

<details>
<summary>💡 Hints</summary>

- This task will require a while loop and a variable to count the sum that is updated throughout the loop.
- Use `int(input())` to read an integer from the user.
- The `0` that ends the sequence should **not** be added to the sum.
- Think about what the condition for the while loop should be.

</details>
