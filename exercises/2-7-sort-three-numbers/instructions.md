# 2.7 Sort Three Numbers (if)

## Task

Please write a program that asks the user to input three integers and prints them in ascending order.

The printed string should have the following format: `"a < b < c"` with `a`, `b` and `c` being the three numbers.

## Examples

> Input: `1`, `2`, `3`
> Output: `1 < 2 < 3`

> Input: `120`, `30`, `5`
> Output: `5 < 30 < 120`

<details>
<summary>💡 Hints</summary>

- The three numbers should be entered with three separate `input()` statements
- The auto-correction is a bit fussy, so please make sure you print the order in the **exact** way described above
- If you already know the `sort` or `sorted` method then that's great — but this task is for practicing `if` statements, so please don't use it
- Make sure you work with integers, not strings. Comparison operators work differently on strings than they do on ints.

</details>
