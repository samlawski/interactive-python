# 1.4 Name Formatting (string operations)

## Task

Please fill in the missing part of the code to format the string `name`.

After the formatting, `name` should fulfill the following requirements:

1. There should **not** be any blank spaces or tabs at the beginning or end
2. Each word in `name` should have its **first letter capitalized**

<details>
<summary>💡 Hints</summary>

- Please change `name` by using **string operations** like `.lower()`, `.upper()`, `.strip()`, `.title()`, etc. Don't just overwrite it with the correct result.
- Please enter your solution between the comment tags that say "your code here".

</details>

<details>
<summary>📚 Useful string methods</summary>

| Method      | Description                              | Example                            |
|-------------|------------------------------------------|------------------------------------|
| `.strip()`  | Remove whitespace from both ends         | `"  hi  ".strip()` → `"hi"`       |
| `.lstrip()` | Remove whitespace from the left          | `"  hi".lstrip()` → `"hi"`        |
| `.rstrip()` | Remove whitespace from the right         | `"hi  ".rstrip()` → `"hi"`        |
| `.lower()`  | Convert to lowercase                     | `"HI".lower()` → `"hi"`           |
| `.upper()`  | Convert to uppercase                     | `"hi".upper()` → `"HI"`           |
| `.title()`  | Capitalize first letter of each word     | `"hello world".title()` → `"Hello World"` |

</details>
