# 6.6 Do What You Want

## Task

Please write a program. **Any program.** Just think of something fun to do with the concepts you've learned.

Use comments to let us know **what** you did and **why**.

<details>
<summary>💡 Ideas</summary>

Not sure what to do? Here are a few suggestions:

- Build a mini address book that stores contacts in a dictionary and lets the user search, add, or remove entries
- Create a program that analyzes a text and prints statistics (number of words, sentences, most common letter)
- Write a to-do list manager where users can add, complete, and view tasks
- Build a simple encryption/decryption program (e.g. Caesar cipher)
- Create a mini inventory system for a shop

</details>

<details>
<summary>📚 What you've learned so far</summary>

- `print()` — to display text
- `input()` — to ask the user for something
- Variables — to store values
- Strings — text in quotes
- Numbers — integers and floats
- Operators — `+`, `-`, `*`, `/`, `**`, `%`
- String methods — `.upper()`, `.lower()`, `.strip()`, `.title()`
- Type conversion — `int()`, `float()`, `str()`
- Comparisons — `==`, `!=`, `<`, `>`, `<=`, `>=`
- Conditionals — `if`, `elif`, `else`
- Functions — `def`, parameters, `return`, default parameters
- Error handling — `try`, `except`
- Lists — `[]`, `.append()`, `.remove()`, `.insert()`, indexing, slicing
- For loops — `for item in list`, `range()`
- Dictionaries — `{}`, key-value pairs, `.get()`, `.keys()`, `.values()`
- While loops — `while condition:`

</details>

Have fun! 🎨
