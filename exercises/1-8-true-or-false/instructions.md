# 1.8 True or False (booleans and operators)

## Task

For each expression below, print whether it evaluates to `True` or `False`.

You should **not** just type `True` or `False` — instead, print the **expression itself** and let Python evaluate it.

## Expressions

Print the result of each of these:

1. `10 > 5`
2. `3 == 4`
3. `7 != 7`
4. `"hello" == "Hello"`
5. `100 >= 100`
6. `2 + 3 == 5`

## Example Output

```
True
False
False
False
True
True
```

<details>
<summary>💡 Hints</summary>

- Comparison operators: `>`, `<`, `>=`, `<=`, `==` (equal), `!=` (not equal)
- `print(10 > 5)` will print `True`
- Remember: Python is case-sensitive — `"hello"` and `"Hello"` are **not** the same

</details>
