# 2.2 Sum of Three (input, coercion)

## Task

Please write a program that asks the user to input three numbers and prints their sum to the command line.

## Example

If the user inputs `"1"`, `"3"` and `"3"`, the sum `"7"` should be printed.

<details>
<summary>💡 Hints</summary>

- Inputting three numbers should be done via multiple `input()` statements
- To calculate the sum you will need to convert the inputted strings to integers using `int()`
- You don't need to concern yourself with error handling — if the user inputs `"Bla"` instead of a number then the program may crash and that is OK for now

</details>
