# 1.2 Dad Jokes (print)

## Task

Please print the following joke to the command line:

> Here's the situation: two windmills are standing on a wind farm.<br>
> One asks: "What is your favorite kind of music?"<br>
> The other replies: "I'm a big metal fan."

Each line should be a **separate line** in the command line.

<details>
<summary>💡 Hints</summary>

- You can create strings either with quotes (`"`) or with ticks (`'`)
- You can add strings together with the `+` operator. An example: `"hell" + "o"` results in `"hello"`
- If your string contains a quote character, you can wrap it with the other kind: `'He said "hello"'`

</details>
