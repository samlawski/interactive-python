# 5.7 Bubble Sort (lists, functions, while, for loops)

## Task

Ready for implementing a more complex algorithm?

Why don't you try your hand at one of the basic sorting algorithms: **Bubble Sort**.

Please write a function `bubble_sort(l)` that gets a list `l`, sorts it and returns it.

A description of how this sorting algorithm works can be found here: [Bubble Sort](https://fullyunderstood.com/pseudocodes/bubble-sort/)

## Examples

> `bubble_sort([1, 12, 3])` → `[1, 3, 12]`<br>
> `bubble_sort([23, 151, 6])` → `[6, 23, 151]`
