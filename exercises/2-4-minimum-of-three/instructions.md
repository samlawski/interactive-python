# 2.4 Minimum of Three (input, coercion, if)

## Task

Please write a program that asks the user to input three numbers and prints the smallest of them.

Please print the exact line `"The minimum is x"` where `x` is the number.

## Examples

- If the user inputs `"1"`, `"3"` and `"6"`, the program should print `"The minimum is 1"`.
- If the user inputs `"13"`, `"7"` and `"1213"`, the program should print `"The minimum is 7"`.

<details>
<summary>💡 Hints</summary>

- Inputting three numbers should be done via multiple `input()` statements
- Do **not** use the `min` function to solve this task — the goal is to practice if-conditionals
- If you want to add together a string and a number you have to convert that number into a string first using `str()`
- You don't need to concern yourself with error handling — if the user inputs `"Bla"` instead of a number then the program may crash and that is OK for now

</details>
