# 4.3 Sum of List (lists, functions, for loops)

## Task

Please write a function `list_sum(l)` that takes one list `l` as input parameter and returns the sum of all numbers **without using the `sum()` function**.

For this task you can assume that `l` is always a list and contains only numbers (ints and floats).

## Examples

> `list_sum([1, 2, 4])` → `7`<br>
> `list_sum([7, 70, 0.7])` → `77.7`

<details>
<summary>💡 Hints</summary>

- You can assume that `l` is always a list and contains only numbers (ints and floats). There is no need to check types or handle exceptions.
- This task will require you to combine a for loop with a variable to keep track of the sum.
- Your task is to **return** the sum, not to print it.
- The goal of this task is to practice iterating over a list and doing something with the elements. That's why the `sum()` function is forbidden.

</details>
