# 3.1 Successor and Predecessor (functions)

## Task

Please define two functions `succ` and `pred`.

Both functions should take one number as input parameter and be defined the following way:

- `succ` should return the **successor** of this number (i.e. the number + 1)
- `pred` should return the **predecessor** of this number (i.e. the number − 1)

## Examples

> `succ(1)` → `2`<br>
> `succ(6)` → `7`<br>
> `succ(33)` → `34`<br>

> `pred(2)` → `1`<br>
> `pred(8)` → `7`<br>
> `pred(33)` → `32`

<details>
<summary>💡 Hints</summary>

- Use the `def` keyword to define a function
- Each function takes one parameter and should `return` a value
- `succ` adds 1 to its parameter, `pred` subtracts 1

</details>
